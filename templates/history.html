How run history works
Every time you click Run All Engines, a snapshot of all results is saved here. Runs with Full Data can be reloaded — clicking Load This Run makes all engine pages (Dashboard, Engine 1–4) show that run's data instead of the latest one. This lets you compare results across different dates or undo the effect of a new run. Runs marked Summary Only were made before this feature was added — only their totals are saved.
